# Установка пакетов

УСТАНАВЛИВАЕМ ТОЛЬКО ЧЕРЕЗ PNPM

## Установка pnpm

```bash
npm i -g pnpm
```

## Установка пакетов

```bash
pnpm i
# ИЛИ
npx pnpm i
```

# Запуск

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
